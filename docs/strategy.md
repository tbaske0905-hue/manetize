# マネタイズ戦略

## フェーズ別目標

### Phase 1（〜1ヶ月目）: 実績ゼロからの脱却
- **目標売上**: 30,000円
- **戦略**: 低単価でも確実に受注 → レビューと実績を積む
- **ターゲット案件**: Excel VBA・データ整理・簡単なスクレイピング
- **1日の目標**: 2〜3案件に提案文を送る

### Phase 2（2〜3ヶ月目）: 単価アップ
- **目標売上**: 100,000円/月
- **戦略**: レビュー実績をもとに高単価案件へ
- **ターゲット案件**: 自動化ツール・継続案件・Python開発
- **継続案件化**: 納品後に月次メンテを提案

### Phase 3（4ヶ月目〜）: 仕組み化
- **目標売上**: 300,000円/月
- **戦略**: ツール系をパッケージ化して定額販売
- **追加収益源**: note有料記事、ツール販売

---

## 案件カテゴリ別戦略

### Excel VBA・マクロ開発
- **単価**: 25,000〜100,000円/件
- **工数**: 1〜5時間（Claude Code活用）
- **勝ちパターン**: 仕様書をそのままClaudeに渡してコード生成 → テスト → 納品
- **提案のコツ**: 「どんなExcel操作でも対応可」「最短翌日納品」

### Webスクレイピング
- **単価**: 10,000〜50,000円/件
- **工数**: 2〜6時間
- **技術スタック**: Python + BeautifulSoup / Playwright
- **注意点**: 利用規約確認、robots.txt確認

### データ変換・整形スクリプト
- **単価**: 5,000〜20,000円/件
- **工数**: 1〜3時間
- **勝ちパターン**: CSV/JSON/Excel間の変換、データクレンジング

### 業務自動化ツール
- **単価**: 30,000〜150,000円/件
- **工数**: 5〜20時間
- **継続性**: 高（メンテ契約に繋がりやすい）

## 提案文の型

```
【受注率を上げる提案文の構成】

1. 課題への共感（1〜2行）
   「〇〇の手作業が多くて大変ですよね」

2. 自分ができること（具体的に）
   「Pythonで△△を自動化するスクリプトを作成できます」

3. 納期・実績
   「最短翌日納品可能。VBA案件XX件の実績あり」

4. 質問（会話を生む）
   「現在のExcelのバージョンと、データ件数を教えていただけますか？」
```

## Claude Code活用フロー

```
1. 案件詳細をコピー
2. Claudeに「この仕様でPythonコードを書いて」と投げる
3. 生成されたコードをClaude Codeでテスト・修正
4. 納品ファイルとして整形
5. tools/ に再利用可能な部分を保存
```
