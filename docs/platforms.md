# プラットフォーム別メモ

## クラウドワークス (crowdworks.jp)

### 特徴
- 日本最大のクラウドソーシング
- タスク形式・プロジェクト形式・時間単価形式
- 手数料: 報酬の5〜20%（報酬が高いほど低率）

### 有力な案件カテゴリURL
- VBA・マクロ: https://crowdworks.jp/public/jobs/category/13
- スクレイピング: https://crowdworks.jp/public/jobs/category/282
- Python: https://crowdworks.jp/public/jobs/search?search[keywords]=Python
- AI・自動化: https://crowdworks.jp/public/jobs/category/283

### 攻略メモ
- 新着順で毎日チェック（人気案件は応募が集中する）
- プロフィールにポートフォリオを必ず掲載
- 最初の5件は低単価でも受注してレビューを集める

## Lancers (lancers.jp)

### 特徴
- クラウドワークスに次ぐ国内2位
- 「パッケージ」機能：自分でサービスを出品できる
- 手数料: 16.5%（一律）

### 有力な案件カテゴリURL
- VBA: https://www.lancers.jp/work/search/system/vba
- スクレイピング: https://www.lancers.jp/work/search/system/scraping
- 自動化: https://www.lancers.jp/menu/tag/自動化

### パッケージ出品戦略
- スクレイピングツール作成「5,000円〜」で出品して集客
- 問い合わせから高単価案件に繋げる

### 攻略メモ
- パッケージ出品（自分からサービスを売る）が差別化しやすい
- 案件スクリーニング自動化ツールが有効（Zenn記事参照）

## 比較表

| 項目 | クラウドワークス | Lancers |
|---|---|---|
| 案件数 | 多い | やや少ない |
| 手数料 | 5〜20% | 16.5% |
| 出品機能 | あり | あり（強い） |
| 初心者向け | ◎ | ○ |

## 競合調査メモ

- スクレイピング案件で上位に出てくるワーカーのプロフィールを参考にする
