# tools/ — 再利用可能なツール集

案件ごとに作成したコードのうち、再利用できるものをここに保管する。

## ディレクトリ構成

```
tools/
├── scraper/          # Webスクレイピング系
├── vba_generator/    # VBA・Excelマクロ生成系
└── proposal_generator/ # 提案文生成系
```

## 使い方

案件を完了したら、汎用的に使える部分をここに移してREADMEに追記する。
「このツールがあるから次の案件も速く納品できる」という状態を目指す。
