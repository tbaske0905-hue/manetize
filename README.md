# manetize

Claude Codeを活用してクラウドワークス・Lancers案件をこなし、マネタイズするプロジェクト。

## 目標

- 月収: 段階的に 3万 → 10万 → 30万円
- 手法: 案件実装をClaude Codeで高速化 → 数をこなす
- 差別化: 即レス・即納品・継続案件化

## クイックスタート

```bash
# バックログを見る
cat projects/backlog.md

# 新規案件を開始する
cp projects/_template.md projects/active/$(date +%Y%m%d)_案件名.md
```
