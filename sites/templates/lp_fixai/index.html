// PROBLEM

AIが作る画面には、
共通のパターンがある

速く動くものができても、見た目がAIらしさを主張してしまう。その原因は3つです。

⬛

余白が均一すぎる

すべてのセクションに同じpadding。情報の重みが伝わらず、どこを見ればいいか分からない画面になる。

🔵

フォントに強弱がない

見出し・本文・注釈のサイズ差が小さい。視線の導線が設計されていないため、読み流される。

🟢

CTAが背景に沈む

ボタンの色がブランドカラーと同化している。最も重要な行動喚起が、画面の中で迷子になっている。

// HOW IT WORKS

3ステップで、
AIへの修正指示が届く

デザインの知識は不要。画面を送るだけで、あなたのAIに渡せる指示を返します。

STEP 01

画面を送る

スクリーンショットまたは URL を送信。LP・Webアプリ・管理画面、どの画面でも対応します。

STEP 02

AIっぽさを特定

座標付きのコメントで問題箇所を明示。「なぜ変えるか」の理由も一緒に返すので、AIへの指示精度が上がります。

STEP 03

そのままAIに渡す

Markdown / JSON / コード形式で納品。Claude Code・Cursor に貼り付けるだけで修正が始まります。
